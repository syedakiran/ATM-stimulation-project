# ATM-stimulation-project
This project is a simple ATM Simulation System developed in C++ using Windows API (Win32). It provides a graphical user interface (GUI) that allows users to perform basic banking operations such as:  Checking account balance  Depositing money  Withdrawing money
Features

Interactive Windows GUI

Deposit functionality

Withdraw functionality with balance validation

Real-time balance checking

Message box notifications

Encapsulated ATM class logic

Simple and clean UI layout

Compatible with Dev-C++
